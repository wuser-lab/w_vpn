# Store release checklist

## Apple

- [ ] Final bundle IDs for app and packet-tunnel extension
- [ ] Network Extension capability and provisioning profiles
- [ ] Sign in with Apple and StoreKit 2 subscriptions
- [ ] App Privacy answers match actual collection
- [ ] Privacy policy and support URLs are live
- [ ] Export-compliance answers reviewed for VPN cryptography
- [ ] Reviewer notes include a usable demo account and connection steps
- [ ] iPhone, iPad compatibility, and macOS screenshots verified

## Google Play

- [ ] Final application ID and release signing key
- [ ] Play Billing subscriptions and backend verification
- [ ] Data Safety form matches actual collection
- [ ] `VpnService` use documented in the store listing
- [ ] In-app prominent disclosure and affirmative consent implemented
- [ ] VPN declaration and short reviewer video submitted
- [ ] Privacy policy and account-deletion flow are live
- [ ] Current target API requirement verified before release

## Both stores

- [ ] No traffic, DNS, or destination logging
- [ ] Independent security review completed
- [ ] Restore purchases and entitlement expiry tested
- [ ] Offline, captive portal, sleep/wake, network-switch, and IPv6 tested
- [ ] Incident response, key rotation, abuse handling, and support runbooks ready
